    Information, instructions and code regarding the Keyes Bluetooth multifunctional car B

    Project maintained by toneby Hosted on GitHub Pages — Theme by mattgraham

    This is the car this site is about: Keyes BT multifunctional car. I bought this car directly from Keyes through Alibaba.com for about $83 including shipping to Sweden, and got it in 12 days. A very good price in my opinion.

    The only problem with it is that it does not come with a manual and I've never played with an Arduino before. So I had to dig around a bit to assemble it. To make it work I had to dig quite a lot. So I decided to make this page to help people out a bit, the resources here are probably useful for other similar cars as well.
